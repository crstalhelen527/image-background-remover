# 📋 QuickBG - 图片背景去除网站 MVP 需求文档

## 🎯 项目概述

**项目名称**: QuickBG - 快速图片背景去除工具  
**定位**: 轻量级、快速、隐私友好的在线图片背景去除服务  
**核心价值**: 用户上传图片 → 调用Remove.bg API → 返回去除背景的图片  
**技术栈**: HTML5 + CSS3 + JavaScript + Cloudflare Workers + Remove.bg API  
**部署**: Cloudflare Workers + Pages  
**存储**: 无服务器存储，纯内存处理  
**API**: Remove.bg API (50张/月免费额度)

## 🛠 技术架构

### 前端架构 (Static Site)
```
Frontend (纯静态)
├── HTML5 (语义化标签 + 现代API)
├── CSS3 (Flexbox/Grid + 响应式设计 + CSS变量)
├── JavaScript (ES6+ Modules)
├── Canvas API (图片预览与处理)
├── File API (文件上传)
├── Fetch API (调用Worker)
└── Service Worker (离线支持)
```

### 后端架构 (Serverless)
```
Cloudflare Worker (无服务器)
├── 接收前端请求 (POST /api/remove-bg)
├── 验证请求参数
├── 调用Remove.bg API
├── 处理API响应
├── 错误处理与重试
└── 返回Base64图片数据
```

### 数据流
```
用户浏览器 → 前端页面 → Cloudflare Worker → Remove.bg API → Cloudflare Worker → 前端页面 → 用户浏览器
    ↓           ↓           ↓           ↓           ↓           ↓
  上传图片    Base64编码   转发请求   专业处理    Base64响应   显示结果
```

### 技术选型理由

1. **Cloudflare Workers**
   - 全球边缘网络，低延迟
   - 免费额度充足（100,000次/天）
   - 无需服务器运维
   - 内置DDoS防护

2. **Remove.bg API**
   - 专业级背景去除效果
   - 50张/月免费额度
   - 简单易用的API
   - 支持多种图片格式

3. **纯前端处理**
   - 用户隐私保护（图片不上传到我们服务器）
   - 减少服务器负载
   - 更好的用户体验

## 🚀 部署方案

### Cloudflare部署配置

#### 1. Worker配置 (`wrangler.toml`)
```toml
name = "quickbg-worker"
compatibility_date = "2024-03-22"
account_id = "your-account-id"
workers_dev = true

[env.production]
route = "quickbg.yourdomain.com/*"
workers_dev = false

[vars]
REMOVEBG_API_KEY = "your-removebg-api-key"
ALLOWED_ORIGINS = "https://quickbg.yourdomain.com"

[build]
command = "npm run build:worker"
```

#### 2. Pages配置 (`_redirects`)
```
/* /index.html 200
/api/* https://quickbg-worker.your-account.workers.dev/api/:splat 200
```

#### 3. 环境变量
- `REMOVEBG_API_KEY`: Remove.bg API密钥
- `ALLOWED_ORIGINS`: 允许的域名（CORS）
- `RATE_LIMIT`: API调用频率限制

### 部署流程

1. **准备阶段**
   ```bash
   # 安装Wrangler CLI
   npm install -g wrangler
   
   # 登录Cloudflare
   wrangler login
   ```

2. **配置阶段**
   ```bash
   # 初始化Worker
   wrangler init quickbg-worker
   
   # 配置环境变量
   wrangler secret put REMOVEBG_API_KEY
   ```

3. **部署阶段**
   ```bash
   # 部署Worker
   wrangler publish
   
   # 部署Pages
   wrangler pages deploy ./public
   ```

4. **域名绑定**
   ```bash
   # 绑定自定义域名
   wrangler route create quickbg.yourdomain.com/* --worker quickbg-worker
   ```

```javascript
// 文件类型验证
const ALLOWED_TYPES = ['image/jpeg', 'image/png', 'image/webp'];

function validateFile(file) {
```

```javascript
// 前端图片压缩
async function compressImage(file, maxWidth = 1200, quality = 0.8) {
  return new Promise((resolve) => {
    const img = new Image();
    img.onload = () => {
      const canvas = document.createElement('canvas');
      const ctx = canvas.getContext('2d');
      
      // 计算缩放尺寸
      let width = img.width;
      let height = img.height;
      if (width > maxWidth) {
        const ratio = maxWidth / width;
        width = maxWidth;
        height = height * ratio;
      }
      
      canvas.width = width;
      canvas.height = height;
      ctx.drawImage(img, 0, 0, width, height);
      
      // 转换为Blob
      canvas.toBlob(resolve, 'image/jpeg', quality);
    };
    img.src = URL.createObjectURL(file);
  });
}
```
