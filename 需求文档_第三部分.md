### 部署步骤

#### 步骤1：初始化项目
```bash
# 创建项目目录
mkdir quickbg && cd quickbg

# 初始化Worker
wrangler init quickbg-worker

# 初始化前端
npm init -y
npm install
```

#### 步骤2：配置Worker
```toml
# wrangler.toml
name = "quickbg-worker"
type = "javascript"
account_id = "your-account-id"
compatibility_date = "2024-03-22"

[env.production]
name = "quickbg-worker-prod"
workers_dev = false
route = "api.quickbg.yourdomain.com/*"

[vars]
REMOVEBG_API_KEY = "your-api-key-here"
ALLOWED_ORIGINS = "https://quickbg.yourdomain.com"

[build]
command = "npm run build"
```

#### 步骤3：设置环境变量
```bash
# 设置API密钥（生产环境）
wrangler secret put REMOVEBG_API_KEY

# 设置允许的域名
wrangler secret put ALLOWED_ORIGINS
```

#### 步骤4：部署Worker
```bash
# 测试部署
wrangler dev

# 发布到workers.dev
wrangler publish

# 部署到生产环境
wrangler publish --env production
```

#### 步骤5：部署前端
```bash
# 构建前端
npm run build

# 部署到Cloudflare Pages
wrangler pages deploy ./dist

# 或使用Pages Dashboard
# 1. 登录 Cloudflare Dashboard
# 2. 选择 Pages
# 3. 创建新项目
# 4. 连接Git仓库或上传文件夹
```

#### 步骤6：配置域名
```bash
# 添加自定义域名到Pages
wrangler pages domain add quickbg.yourdomain.com

# 添加路由到Worker
wrangler route create api.quickbg.yourdomain.com/* quickbg-worker
```

#### 3. 使用量监控
```javascript
// 记录API使用
class UsageTracker {
  constructor() {
    this.dailyLimit = 50;
    this.monthlyLimit = 1500;
  }

  async trackUsage(credits = 1) {
    const today = new Date().toISOString().split('T')[0];
    const month = new Date().toISOString().substring(0, 7);
    
    // 更新本地存储
    const daily = parseInt(localStorage.getItem(`usage_daily_${today}`) || '0');
    const monthly = parseInt(localStorage.getItem(`usage_monthly_${month}`) || '0');
    
    localStorage.setItem(`usage_daily_${today}`, daily + credits);
    localStorage.setItem(`usage_monthly_${month}`, monthly + credits);
    
    // 检查限制
    if (daily + credits > this.dailyLimit) {
      throw new Error('今日使用次数已达上限');
    }
    
    if (monthly + credits > this.monthlyLimit) {
      throw new Error('本月使用次数已达上限');
    }
    
    return {
      daily: daily + credits,
      monthly: monthly + credits,
      dailyRemaining: this.dailyLimit - (daily + credits),
      monthlyRemaining: this.monthlyLimit - (monthly + credits),
    };
  }
}
```


**问题2: Worker部署失败**
```bash
# 回滚到上一版本

# 或重新部署

```

**问题3: 前端资源404**
```bash
# 清除缓存
wrangler pages purge-cache

wrangler pages deploy ./dist
```

**场景: Cloudflare服务中断**
1. 启用备份域名
2. 切换到备用CDN
3. 通知用户服务维护

**场景: Remove.bg API不可用**
1. 启用降级模式（基础颜色去除）
2. 显示维护通知
3. 联系API提供商

**场景: 数据丢失**
1. 从Git恢复代码
2. 从备份恢复配置
3. 重新部署服务
